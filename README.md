# 🧮 Tip Calculator

A clean, dark-themed tip calculator web app — built as a single HTML file. No frameworks, no dependencies, no build step.

[![Live Demo](https://img.shields.io/badge/Live%20Demo-%E2%86%92-7ef0b4?style=for-the-badge&labelColor=0f1117)](https://div10000.github.io/tip-calculator/tip-calculator.html)

---

## ✨ Features

- **Bill Amount Input** — Enter any bill amount with a `$` prefix field
- **Quick Tip Presets** — One-click buttons for 5%, 10%, 15%, and 20%
- **Custom Tip %** — Type any tip percentage; automatically deselects the presets
- **People Splitter** — `+` / `−` stepper to set the number of people splitting the bill
- **Live Calculations** — Results update instantly as you type, no submit needed
- **Three Key Outputs:**
  - 💰 Tip amount per person
  - 🧾 Total bill (with tip included)
  - 👤 Each person's total share
- **Reset Button** — Clears all fields and results in one click
- **Fully Responsive** — Two-column layout on desktop, single-column on mobile

## 🚀 Usage

No installation needed. Just open the file in any browser.

```bash
# Clone the repo
git clone https://github.com/div10000/tip-calculator.git

# Open directly in browser
open tip-calculator.html
```

Or simply [download the HTML file](https://github.com/div10000/tip-calculator/blob/main/tip-calculator.html) and open it locally.
